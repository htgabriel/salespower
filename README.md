# Sales Power

## Scripts acessíveis
```bash
  npm run deploy
```
Roda o app.\
A porta padrão é definida como `3000`.\
\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.


## Para realização dos testes
- Abra a coleção do Postman 
- Envie a requisição `Login`
- Salve o token na caixa Token nas configurações da coleção
- Envie a requisição `Get Info (Authenticated)`
